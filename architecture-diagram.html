Input Sources

🛡
Detection Engine
presidio + spaCy NER
  • 20+ entity types (EMAIL, PERSON, IP, PHONE, CREDIT_CARD, SSN...)
  • Overlap resolution (highest confidence wins)
  • Schema-aware: per-field scanning for JSON
  • Confidence scores per entity
  • Custom recognizer extensibility
Processing Pipeline
📥 Input
Text or structured JSON enters the pipeline
▼
🔍 Detect
presidio + spaCy NER identifies PII entities with confidence scores
▼
🧹 Sanitize
Policy-driven: redact, pseudonymize, or allowlist per entity type
▼
📊 Score
Privacy risk 0.0 (NONE) → 1.0 (CRITICAL) based on density + confidence
▼
📝 Audit
SQLite log: every action recorded with timestamp and policy reference
